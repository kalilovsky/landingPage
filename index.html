à Propos de Moi

J'ai toujours été un passionné de l'informatique et la programmation. Déja, dés l'âge de 11 ans (1996) je me suis mis à la programmation d'abord avec le langage QuickBasic, puis Visual basic pour arriver au C++ et au C Sharp en utilisant DirectX pour créer des jeux. Ensuite, durant ma carrière professionnelle, j'ai mis en application toutes les connaissances acquises pour développer des outils qui me permettaient de bien piloter mes différentes entreprises. Aujourd'hui j'ai fait de ma passion mon métier, j'ai suivi une formation intensive de 7 mois, au sein de l'école Simplon à Marseille, qui m'a permis d'acquérir les outils nécessaires pour devenir un développeur Web Fullstack.

Mes Compétences

Les principales technologies que je maitrise.

React JS

Que ce soit en fonctions ou en classes, les hooks, le JSX, Redux ...etc. React Js est un puissant outil qui me permet d'avancer vite et de délivrer un résultat optimal, avec ou sans Typescript.

Vue 3

En Compositions API ou Options API, Pinia ou Vuex, je prends toujours du plaisir à utiliser Vue Js, avec ou sans Typescript. Il m'est souvent arrivé de l'utiliser avec le puissant Primevue.

Symfony 5

Aprés avoir fais du PHP en MVC, et des requêtes SQL, j'ai découvert Symfony 5, il me permet de ne plus penser aux bases d'une application, mais de me concentrer sur la résolution des problèmes réels du projet.

HTML5 CSS3

Que ce soit le HTML ou le CSS3, je les maitrise assez bien pour livrer un front respectable.

JAVASCRIPT

Les bases JS, JQuery et le DOM sont bien maintrisées, ainsi que quelques API WEB tel que Canvas,IndexedDB ,Fetch, AJAX, Object, Promises, Proxy,Observers ...etc.

PHP

MYSQL

De la création d'une base de données selon la méthose Merise, ou que ce soit INSERT, SELECT, INNER JOIN, VIEWS....etc MySQL n'a plus aucun secrets pour moi.

WORDPRESS
